健身勇者 RPG 4.0

新功能：
- 戰鬥動畫：角色衝刺、斬擊、怪物受傷與傷害數字
- 五張地圖、五個 Boss 與主線劇情
- NPC 指引
- 商店與金幣購買裝備
- 裝備可增加攻擊力
- 技能樹與技能點
- 等級、XP、Combo、連續訓練天數
- 自訂新增與刪除健身任務
- 每日任務自動重置
- 戰鬥紀錄與 JSON 存檔匯出

GitHub 更新方式：
1. 解壓縮 ZIP。
2. 將以下五個檔案上傳並覆蓋 repository 根目錄：
   index.html
   manifest.webmanifest
   service-worker.js
   icon.svg
   README.txt
3. Commit changes。
4. 等候 GitHub Pages 約 1 分鐘。
5. Safari 重新整理；若仍顯示舊版，完全關閉主畫面 App 後再開啟。
